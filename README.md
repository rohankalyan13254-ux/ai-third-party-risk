# AI for Third-Party Risk Management

This repository explores how AI can be applied to improve third-party risk assessment workflows, particularly in areas like questionnaire analysis, risk summarization, and documentation standardization.

## Why this matters

Traditional vendor risk assessments are:
- Manual and time-consuming
- Inconsistent across reviewers
- Heavily documentation-driven

AI can help standardize decision-making and reduce repetitive effort.

## What this repo includes

- Sample vendor risk questionnaire
- Prompt templates for risk analysis
- Example AI-generated risk summaries

## Example Prompt

You are a cybersecurity risk analyst.

Analyze the following vendor response and provide:
1. Key risks identified
2. Missing or weak controls
3. Suggested risk rating (Low / Medium / High)
